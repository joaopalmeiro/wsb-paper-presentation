# wsb-paper-presentation

Material for the presentation on the Width-Scale Bar Charts (Höhn et al., 2020).

## Quickstart

- `pipenv install`
- `pipenv shell`
- `pipenv run jupyter notebook`

## Commands

- Restart Jupyter Notebook: `Cmd+R`
- Enter/Exit RISE slideshow: `Option+R`
